# like-bot

## Create vertual environment

```bash
python -m venv venv
```

## Environment activate

### For Mac or Linux

```bash

source venv/bin/activate
```

### For Windows

```bash

source venv/Scripts/activate
```

## Install required libraries

```bash
pip install -r requirements.txt
```
